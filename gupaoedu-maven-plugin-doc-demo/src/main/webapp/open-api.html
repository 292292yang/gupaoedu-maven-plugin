公共状态码说明
状态值 描述
0 服务器内部异常
1 返回结果正常
2 返回结果失败
3 请求超时
4 不允许访问的接口
9 请求参数错误
98 未授权不允许访问
99 未登录
返回结果说明
字段名称 字段类型 说明 示例
success 布尔型 是否成功 判断此次结果调用返回是否正确
errorCode 字符串 错误码 错误码消息:每个错误消息对应一个错误码
single Object 返回对象类型结果 返回结果，可为任意单个对象
list List 返回集合类型结果 返回集合，可为任意多个对象
